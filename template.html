Redirect file template

Copy this into a .html file, replace the placeholder with your markdown, and opening the file lays it out in Dense Printer, ready to print. Your content never leaves the browser — it travels in the URL fragment.

<!doctype html><meta charset="utf-8"><title>My Document</title>
<!--
  Dense Printer redirect file: opening this in a browser lays the content
  below out as dense printable pages at
  https://avi-perl.github.io/dense-printer/. Nothing is uploaded; the
  content travels in the URL fragment.

  AI agents: read https://avi-perl.github.io/dense-printer/llms.txt to learn
  how to use this file and everything else the tool can do.
-->
<script src="https://avi-perl.github.io/dense-printer/open.js"></script>
<script type="text/markdown">
# Your title here

Replace everything in this block with your markdown, verbatim. Don't
indent it.
</script>

The <title> becomes the document name. Layout falls back to the site's defaults; to pin any setting, add data-* attributes to the loader script tag — e.g. data-size="8" data-cols="2" data-autofill="1" (any URL API parameter works). Raw HTML instead of markdown: drop the text/markdown block and put the HTML straight in the body.
