# Markdown Page Printer

Pack markdown into dense, print-ready 8.5×11 pages — tune columns, type size, margins, and squeeze, then Print/PDF. A static, client-side-only site; nothing leaves your browser.

**Open it:** https://avi-perl.github.io/dense-printer/ · or just open `index.html`.

**For AI agents & automation:** the app has a URL API — pass base64 markdown and settings in the URL fragment (`#md=<base64>&size=8&cols=3`) and it renders fully paginated, ready for headless print-to-PDF. See [llms.txt](llms.txt) for the full parameter list and copy-paste recipes.
