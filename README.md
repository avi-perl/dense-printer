# Markdown Page Printer

Pack markdown into dense, print-ready 8.5×11 pages — tune columns, type size, margins, and squeeze, then Print/PDF. A static, client-side-only site; nothing leaves your browser.

**Open it:** https://avi-perl.github.io/dense-printer/ · or just open `index.html`.
